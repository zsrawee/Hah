# Random Hadith Generator
Use this skill when the user wants a random hadith with full details.
The skill will fetch a random hadith from the database and display it beautifully.

## Steps
1. Call `/api/random` to get a random hadith
2. Display the result with:
   - Arabic text (with diacritics)
   - English translation
   - Grade badge (Sahih/Hasan/Da'if)
   - Collection name and book reference
   - Chain of narration (Sanad)
